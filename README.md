# PowerJob Documentation

This is the official documentation website for [PowerJob](https://github.com/PowerJob/PowerJob), built with [VitePress](https://vitepress.dev/).

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm docs:dev

# Build for production
pnpm docs:build

# Preview production build
pnpm docs:preview
```
