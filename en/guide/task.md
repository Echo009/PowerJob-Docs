## Overview

A task is the basic scheduling unit in PowerJob, containing task configuration, execution logic, and scheduling strategy.

## Task Properties

### Basic Configuration
- Task Name
- Task Description
- Execution Type
- Processor Information

### Schedule Configuration
- Schedule Strategy
- CRON Expression
- Execution Frequency

### Runtime Configuration
- Timeout
- Retry Count
- Concurrency
